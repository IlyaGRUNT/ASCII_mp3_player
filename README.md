# ASCII_mp3_player
simple ASCII mp3 player in CMD Windows


For correct work ffmpeg(https://ffmpeg.org/) need to be installed in c:/ffmpeg/



Before start put any music in mp3 format into a floder with downloaded main.py

# Control buttons
P - pause music

U - unpause music

W - increase volume

S - decrease volume

Q - previous track

E - next track

Z - exit
